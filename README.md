For FreeCAD 1.1.1 and 26.3

A simple add-on that opens a second window containing Property View, Select View, Report View, and Python Console, which can be placed on a second monitor.
Nothing happens when you first launch FreeCAD. Restarting FreeCAD will open the second window with the widgets inside.
Right-clicking on the document tab bar brings up a menu with two options: Clone view and Four View Clone
The first option opens an additional view of the document, while the second opens a window with four views: TOP LEFT FRONT and ISOMETRIC.
Added support for Techdraw and Spreadsheet documents

Tested only with OpenTheme Dark.
There are known bugs, this is only a test :)
